HTML Tutorial

Що таке HTML?

HTML — це мова розмітки, яка використовується для створення веб-сторінок.

Основні терміни

Тег — це елемент HTML, який визначає структуру сторінки.

Наприклад, тег <p> використовується для створення абзацу.

Приклад простої сторінки


<html>
  <body>
    <h1>Hello World</h1>
    <p>Це моя перша сторінка</p>
  </body>
</html>

Форматування тексту

За допомогою тега <span> можна виділяти текст або змінювати його стиль.

Інші приклади

HTML дозволяє створювати кольоровий текст та курсив.

Чому важливо вивчати HTML?

HTML є основою для створення будь-якого сайту.

Висновок

Вивчення HTML — перший крок у веб-розробці.
